## Reporting a Vulnerability

**Do not open a public issue.**

If you discover a security vulnerability in Diamond, please report it privately via GitHub Security Advisories:

**Report here:** https://github.com/rsperko/diamond/security/advisories/new

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
